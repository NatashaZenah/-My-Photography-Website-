## Project Description 
This is a photography-themed website inspired by the work of Steve McCurry

The website shows:

-A gallery of selected photographs taken by Steve McCurry

-A simple navigation structure
-descriptions explaining the importance of each photograph

---
## Technologies Used
-HTML

-CSS

---
## Installation Guide
1.Download the project files into a single folder

2.Ensure all HTML files are in the same directory

3.Ensure the css file "Photography.css" is linked to the HTML file

4.Confirm that all the image files are correctly named and can be opened

5.Open "Photography.html" by going live
